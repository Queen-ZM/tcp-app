# Overview

As a software developer I am trying to make the life of theh people easier by creating applications that will allow that to happen. I have found free courses on line as well that have been helpful in my learning journey.

The server listens for incoming client connections and manages communication between connected users. Clients can send chat messages, submit requests, transfer text files, and disconnect from the server when finished. The application also includes error handling to improve reliability and provide a better user experience.

I chose to create a TCP Client-Server Chat Application because it allowed me to learn and apply important networking concepts in a practical way. Through this project, I was able to gain hands-on experience with the OSI Model, TCP communication, sockets, ports, JSON messaging, and client-server architecture. These are fundamental skills used in software development, networking, and cybersecurity.
